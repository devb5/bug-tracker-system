# Bug Tracking & Project Management System

## Overview
A web-based Bug Tracking and Project Management System developed during internship to manage software projects, track bugs/issues, assign tasks, and monitor workflow efficiently.

---

## Features

- User Authentication
- Role-Based Access Control
- Project Management
- Bug Tracking
- Task Management
- Dashboard Analytics
- Dark Mode
- Activity Logs
- PDF Export
- Responsive UI
- Profile Management

---

## Technologies Used

### Frontend
- HTML
- CSS
- Bootstrap 5
- JavaScript

### Backend
- PHP

### Database
- MySQL

---

## Modules

- Admin Module
- Developer Module
- Tester Module
- Dashboard Module
- Activity Logs Module

---

## Installation

1. Install XAMPP
2. Move project folder to htdocs
3. Import database in phpMyAdmin
4. Start Apache and MySQL
5. Open localhost/project-folder
